## Edge Add-ons Store Description (250–10,000 chars)

**Summary (short):**
Extract, edit, and export color palettes from any website — live, in real time.

**Full Description:**

PaletteLive lets you extract every color from any website, edit them live with instant preview, and export the result in six professional formats — all without reloading the page.

**How It Works:**

1. Visit any website and click the PaletteLive icon in your toolbar.
2. The extension scans the entire page — including Shadow DOM, CSS variables, pseudo-states, and keyframes — and displays every color found as interactive swatches.
3. Click any swatch to open the side panel editor with a full color picker, a live WCAG contrast checker (AA/AAA for normal and large text), and a real-time preview toggle.
4. Changes appear instantly on the page. Toggle "Before / After" to see a split-screen comparison.
5. Export your palette in CSS Variables, JSON Design Tokens, Tailwind Config, CMYK, CIE LAB, or OKLCH format — to clipboard or file.

**Key Features:**

• Live color editing — changes appear on the page in real time, no reload needed.
• Full-page color extraction — scans computed styles, stylesheets, Shadow DOM, CSS custom properties, pseudo-states, and keyframes.
• CIEDE2000 color clustering — groups visually similar colors with a configurable perceptual threshold slider.
• Six palette harmony generators — Monochromatic, 60-30-10, Analogous, Complementary, Split-Complementary, and Triadic.
• WCAG contrast checker — displays contrast ratio, AA/AAA rating, and pass/fail for normal and large text.
• Automatic text contrast enforcement — after any color change, re-checks all text and adjusts foreground colors to meet WCAG AA.
• Color dropper — click any element on the page to pick its color with a crosshair overlay and floating preview.
• Import palettes from file or clipboard — auto-maps imported colors onto the page preserving contrast relationships.
• Six export formats: CSS Variables, JSON Tokens, Tailwind Config, CMYK, CIE LAB, OKLCH.
• Heatmap overlay — outlines every colored element and shows a frequency chart of the page's color usage.
• Before/After comparison — animated split view with draggable divider.
• Vision simulation — preview your page through Protanopia, Deuteranopia, Tritanopia, or Achromatopsia filters.
• Per-domain persistence — saved palettes reapply automatically when you revisit a site.
• SPA-aware — detects route changes and reapplies overrides after navigation.
• Full undo/redo history for all color changes.
• Dark, light, and auto theme for the popup itself.

**Privacy:**

All data stays on your device. PaletteLive makes zero network requests — no analytics, no telemetry, no external servers. See our full privacy policy for details.

**For Designers & Developers:**

PaletteLive is built for anyone who works with color on the web — UI/UX designers auditing accessibility, front-end developers tweaking design tokens, brand managers checking consistency, or students learning color theory. Extract any site's palette, experiment with changes live, and export production-ready code.

## Opera Add-ons Full Description

PaletteLive is a color palette tool that scans any webpage, displays every color as interactive swatches, and lets you edit them live — changes appear instantly on the page without reloading.

**How to use PaletteLive:**

1. Click the PaletteLive icon in your toolbar on any website.
2. The popup shows every color extracted from the page, grouped by type (background, text, border, accent).
3. Click any color swatch to open the editor panel with a color picker and live WCAG contrast checker.
4. Your changes appear on the page immediately. Use the "Before / After" toggle to see a side-by-side comparison.
5. When you're happy, export the palette via the Export menu — choose from CSS Variables, JSON Tokens, Tailwind Config, CMYK, CIE LAB, or OKLCH.

**What it looks like:**

The popup displays a clean grid of color swatches with hex values. A clustering slider merges similar colors for a cleaner view. The side panel editor shows a large color picker, the current and new color side by side, and a WCAG contrast grid showing AA/AAA pass/fail status. The heatmap overlay outlines every colored element on the page with frequency bars.

**Features:**

• Full-page color extraction including Shadow DOM, CSS variables, and pseudo-states.
• Live editing — colors change on the page in real time.
• CIEDE2000 perceptual color clustering with adjustable threshold.
• Six palette harmony generators (Monochromatic, 60-30-10, Analogous, Complementary, Split-Complementary, Triadic).
• WCAG AA/AAA contrast checker and automatic text contrast enforcement.
• Color dropper for picking any element's color.
• Import palettes from file or clipboard with intelligent auto-mapping.
• Six export formats: CSS Variables, JSON Tokens, Tailwind Config, CMYK, CIE LAB, OKLCH.
• Heatmap, Before/After comparison, and vision simulation (color blindness preview).
• Per-domain persistence — palettes reapply automatically on revisit.
